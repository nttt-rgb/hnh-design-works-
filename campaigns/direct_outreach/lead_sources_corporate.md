# 企業向けリード発掘ソース一覧

担当: Scout Agent
作成日: 2026-04-10
対象: HP無し or HP古い小規模企業・個人事業主（年商数千万〜数億の中小）

> Places API枠が少ない期間は、以下のソースを **Web検索ベース** で活用する。
> Tavily/WebSearch + WebFetch の組み合わせで完結する。

---

## 優先度S: 即実行可能・高確率

### 1. Instagram ハッシュタグ検索
**特徴:** 小規模事業者のbio にHPリンクがない/Linktreeだけのケースが多い → HP無し候補

| 業種 | ハッシュタグ | 狙い目 |
|------|-------------|--------|
| パーソナルジム | `#東京ジム` `#渋谷パーソナルジム` `#港区パーソナル` | フォロワー500〜5000の個人経営 |
| ピラティス | `#ピラティス東京` `#恵比寿ピラティス` | 新規開業・単独スタジオ |
| 整体院・鍼灸 | `#整体東京` `#鍼灸渋谷` | 院長1人の小規模院 |
| 美容室 | `#東京美容室` `#渋谷美容室` | 1〜3席の独立店 |
| ネイルサロン | `#ネイルサロン東京` | 個人ネイリスト |
| クリニック | `#美容クリニック東京` | 新規開業 |
| 英会話・スクール | `#英会話東京` `#プログラミングスクール` | 個人講師・小規模教室 |

**確認項目:**
- [ ] bio に公式HP URL があるか
- [ ] あれば `WebFetch` でHPを取得し、スマホ対応/SSL/デザイン年代を確認
- [ ] ない or Linktree のみ → リード候補

### 2. Google 検索 — 業種別 × エリア
```
"[業種] [エリア]" 口コミ
"[業種] [エリア]" おすすめ 2026
"[業種] [エリア]" ランキング
```
実在する店舗名リストを抽出 → 各店舗を個別に `"[店舗名]" 公式サイト` で検索 → HP無しの店舗を特定

## 優先度A: 中効率・高品質

### 3. 求人サイトの小規模企業（Indeed / エンゲージ / Wantedly）
**重要:** Wantedlyはスクレイピング禁止（利用規約違反）。**手動閲覧のみ**で、各企業の公式HPリンク欄を確認する。
**Indeed / エンゲージは公開情報なので OK。**

**手順:**
1. Indeed で「東京 スタートアップ 従業員10人以下」等で検索
2. 各求人に掲載されている会社URLを確認
3. URLが `indeed.com/cmp/...` しかない → 公式HP無し候補
4. あれば `WebFetch` でチェック

### 4. 業界別無料サブスクのスタートアップリスト
- **STARTUP DB** - スタートアップ情報を無料閲覧可能
- **BizReach** - 企業情報ページ（ログイン必要）
- **J-Net21** - 中小企業庁運営、法人情報提供

### 5. 法人番号公表サイト（国税庁・完全無料・合法）
https://www.houjin-bangou.nta.go.jp/
- 全法人データ公開、API提供あり
- エリア・設立時期で絞り込み可能
- 設立3年以内の新しい法人 → HP無し確率高い

## 優先度B: 時間かかるが確実

### 6. 地域商工会・商工会議所の会員名簿
- 東京商工会議所: https://www.tokyo-cci.or.jp/
- 各区商工会: 区のWebサイトから辿れる
- 会員企業一覧が公開されている場合あり
- 小規模製造・サービス業が多い

### 7. ビジネスマッチングサイト
- **Linked Startup** - スタートアップ募集
- **みんなの事業計画書** - 起業家コミュニティ
- **クラウドファンディング完了プロジェクト** (CAMPFIRE, Makuake) - 成功したスモールビジネスは次にHPが欲しい

### 8. Twitter/X 経営者アカウント
```
from:(経営者ハッシュタグ) "会社設立しました"
"会社設立" "税理士紹介" 東京
"開業しました" 2025 OR 2026 東京
```

## 収集データフォーマット（scout_websearch.js 入力用）

```json
[
  {
    "name": "株式会社〇〇",
    "category": "IT企業",
    "area": "渋谷",
    "email": "info@example.com or 空文字",
    "phone": "03-xxxx-xxxx or 空文字",
    "rating": "",
    "reviews": "",
    "hp_status": "なし or 古い（スマホ非対応）",
    "instagram": "https://www.instagram.com/xxx/",
    "note": "Web検索で発見。Instagram フォロワー約1200、bio にHPリンクなし",
    "status": "未送信",
    "source": "websearch"
  }
]
```

保存して `node scripts/auto_pipeline/scout_websearch.js <leads.json>` で `target_list.csv` に追記される。

## 1日の目標ワークフロー

**月目標: 月12件受注 → 日20件アプローチ必要（返信率2%、成約率20%想定）**

| 時間 | アクション | 成果物 |
|------|-----------|--------|
| 30分 | Instagram ハッシュタグで10業種探索 | リード候補50件 |
| 30分 | 各候補のbio確認 + HP有無チェック | HP無しリード20〜30件 |
| 15分 | leads.json 作成 + scout_websearch.js 実行 | target_list.csv 追記 |
| 60分 | 追記した20件に営業メール/DM送信 | 送信済み20件 |
| **計 2h15m** | 日20件営業 | |

これを週5日回せば **週100件 = 月400件** のアプローチ。返信率2%で **月8件の商談**、成約率25%で **月2件受注** → HNH目標 ¥200万/月には届かないが、確実なベース。

**リード発掘を50件/日に倍増できれば月4件受注、100件/日で月8件受注** = ¥130-200万/月。業種拡張と並列チャネル（ココナラ・ジモティー・クラウドワークス）で上積みする。

## 注意事項

1. **Wantedlyスクレイピング禁止** — 利用規約違反。手動閲覧のみ。
2. **特定電子メール法遵守** — 公開されているメールアドレスのみ使用、「営業お断り」表記の除外、オプトアウト明記
3. **Instagram DMは控えめ運用** — 1日30件程度にしないとアカウントBAN リスク
4. **個人情報の取り扱い** — 取得した個人情報は営業目的のみに使用
