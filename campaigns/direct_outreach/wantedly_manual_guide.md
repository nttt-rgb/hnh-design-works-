# Wantedly 手動リード発掘ガイド

担当: Scout Agent
作成日: 2026-04-10

> ⚠️ Wantedlyの利用規約ではスクレイピング・自動収集が明確に禁止されている。このガイドは **「手動ブラウズで企業を見つける」** ところまでをWantedlyで行い、**実際の連絡先取得はGoogle検索で別途行う** という前提で設計されている。

---

## 目的
Wantedly掲載企業のうち、**採用活動を活発にやっているがHPが無い or 古い** 企業を発掘する。
このような企業は Web制作需要が非常に高い：
- 採用に力を入れている → 求職者に向けた公式サイトが必須
- HP無し → 求職者が応募をためらう → 採用効率が落ちている
- **「採用強化のための公式サイト」という刺さるフック** が使える（corporate_email_template.md のパターンB）

---

## 手順（1セット 15分）

### ステップ1: Wantedly にアクセス
1. https://www.wantedly.com/projects を開く
2. ログインしてもしなくてもOK（未ログインでも閲覧可能）

### ステップ2: フィルタ設定
- **勤務地**: 東京
- **募集職種**: エンジニア / デザイナー / 営業 / マーケ 等（絞り込みたい業種）
- 求人タイプは「中途」でも「インターン」でも良い（企業を見つけるのが目的）

### ステップ3: 各企業ページを巡回
気になる企業をクリック → 企業ページへ移動。以下を確認：

1. **「ウェブサイト」欄**
   - 空 → **HP無し候補**（最有力）
   - Wantedlyページ自体がリンク → HP無し候補
   - Google Sites / Notion / Wixの簡易ページ → **古い候補**
   - 本格的な独自ドメインのHP → 対象外（スコアリング後リデザイン提案候補）

2. **「会社規模」欄**
   - 1〜10人 / 11〜30人 → 狙い目（小規模ほどHP無し率が高い）
   - 30人以上 → HP持ってる確率高いのでスキップ

3. **「業種」欄**
   - IT・Web系 → 一見HPありそうだがスタートアップは意外となし
   - 士業・コンサル・デザイン → HP必要な業種なのにない場合あり
   - 建築・不動産・医療 → HP無い率が高い

### ステップ4: 候補企業の名前をメモ
以下のフォーマットでメモ：
```
- 株式会社〇〇（Wantedly URL: https://...）
- 業種: IT・SaaS
- 所在地: 東京都渋谷区
- 会社規模: 1〜10人
- HP欄: 空
- 注記: 採用中（エンジニア募集）
```

### ステップ5: Google検索で連絡先取得
メモした企業名をGoogle検索 → 公式サイトが本当にないか最終確認 → 見つからなければ以下の経路で連絡先探索：

1. **Google検索**: `"株式会社〇〇" お問い合わせ` or `"株式会社〇〇" メール`
2. **法人番号公表サイト**: https://www.houjin-bangou.nta.go.jp/ で会社名を検索 → 住所確認
3. **Eight / Sansan 名刺検索**: 役員の名刺があれば直接連絡
4. **Google Maps**: 会社名で検索 → 電話番号 or 公式サイトリンク

### ステップ6: target_list.csv に追加
取得した情報を `leads.json` にまとめて：
```json
[
  {
    "name": "株式会社〇〇",
    "category": "IT・SaaS",
    "area": "渋谷",
    "email": "info@example.com",
    "phone": "03-xxxx-xxxx",
    "hp_status": "なし",
    "instagram": "",
    "note": "Wantedlyで採用中・エンジニア募集・1-10人規模",
    "status": "未送信",
    "source": "wantedly_manual"
  }
]
```

`node scripts/auto_pipeline/scout_websearch.js leads.json` で追記。

### ステップ7: 営業メール送信
`corporate_email_template.md` の **パターンB**（採用中HP無し企業向け）を使って営業メール作成 → 送信。

---

## 1日の目標
- **目標**: 15分 × 2セット = 30分で 20件 の候補発掘 → 10件の連絡先取得 → 10件送信
- **週5日運用**: 週50件 / 月200件
- 返信率2% → 月4件の商談 → 成約率25% → 月1件受注（平均¥150k）

Instagram DM（月400件）と合わせれば **月12件受注の目標** に近づく。

## 注意事項

### Wantedly利用規約遵守
1. **スクレイピング禁止** — ボット・自動化ツールを使わない。手動ブラウズのみ
2. **Wantedly上のメールアドレスを使わない** — Wantedlyアカウントメッセージ機能経由の情報は使用禁止（営業規約違反）
3. **あくまで「企業を見つける」だけ** — 連絡先取得は必ずWantedly外（Google検索・法人番号サイト・公式メール）から行う

### 特定電子メール法遵守
4. 公開されているメールアドレスのみ使用
5. 「営業お断り」記載のある企業には送らない
6. オプトアウト文言を必ず明記

### Wantedly上の営業禁止
7. **Wantedlyのメッセージ機能で営業メッセージを送らない** — 即BANリスク + 利用規約違反
8. Wantedlyは「採用プラットフォーム」であり、「営業プラットフォーム」ではない
