Case Study 01 Light Plan — ¥110,000 高級江戸前寿司 銀座

鮨 黒政 SUSHI KUROMASA

銀座6丁目の路地裏、看板も出さない8席の鮨屋。
本物の格を、Webで再現する。

Light Plan
01

Challenge

25年の格を、一見の方にも伝える。

銀座6丁目の路地裏、看板も出さない8席の鮨屋。常連客のみで成り立ってきた25年。

息子への代替わりを機に「でも一見の方にも、店の格を伝えたい」と相談を受けた。ただし、安っぽいサイトでは逆効果。本物の格を、Webで再現する必要があった。

02

Solution

他は「写真の多用」。私たちは「文字で語る」を選んだ。

他の高級鮨店のサイトを20件以上分析。共通していたのは「写真の多用」と「英語対応」だった。ただ、黒政の世界観は静謐。それを真似ては凡庸になる。

選んだのは逆の方向 ― 写真は最小限、文字で語る。

写真は1枚だけ。カウンターの大将の手元のみ。「本物は雄弁にならない」を体現。

03

Design Details

静謐を成立させる、4つの細部。

01

筆文字フォント

MYueShen を採用し、看板の毛筆を画面上で再現。本文は明朝で「読ませる静けさ」を。

02

墨のにじみフィルタ

SVG filter (turbulence + displacement) で、和紙にしたためた文のような質感をブラウザネイティブで実装。

03

カスタムカーソル

暖簾の紋を模した形状を Custom Cursor API で描画。常連の入口を抜ける所作を再現。

04

沈み込むパララックス

スクロールに連動して、ページが深く沈み込む。カウンターに座り、店の空気に身を預ける動線。

04

Result

公開後3ヶ月の数字。

予約問い合わせ

15件 → 月42件
3ヶ月平均

インバウンド予約比率

3% → 28%
対前期比

平均客単価

変化なし

高単価客の流入を維持

━━━━━━━━━━━━━━━━━━

打ち合わせもせず、写真もほとんど送っていないのに、店の空気が画面から伝わってくる。
— 鮨 黒政 二代目
05

Tech Stack

no-framework, all-handcrafted.

HTML/CSS/JS no framework GSAP SVG Filters Custom Cursor API Google Maps Embed
